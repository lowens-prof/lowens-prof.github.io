Breakdown

How does the program run:

  • 5 Sessions - 1.5 hours each. The sessions are completely catered to your schedule. Weekends are more available.

Where does it take place:

  • Zoom, as it has features that make it easy for dictating instructions via Remote Control

Hardware needed:

  • Any computer that has access to chrome browser. We utilize a game engine called Construct 3. Construct 3 has a paid version, however, it is not required

What does the program entail:

  • Starting with the basics of using a game engine. Learning programming terms and fundamentals found in traditional programming languages. Making classic games with a large list of suggestions that the student can choose based on their intrest. The slideshow above demonstates a small sample of games. Other topics include, game development, level design, animations, variables, conditions and actions, coding etiquette.

What is the outcome:

  • Aside from learning programming terms and fundamentals, the student would create their own game based on their passion. Which can be shared amonst their friends and family.

Coding experience needed?

  • None, no prior coding experience is to be expected

Cost:
